# i3-double-monitor-fix
bash script to fix i3 window manager double monitor problems: detect monitors then fix display using xrandr

> Monitor ID's like `eDP`  and `HDMI-A-0`  can be different in your side 

### How to use
- It can be used in startup by modifying `~/.xinitrc` with the content of the script.
- Every time i unplug or plug the second monitor, i run this script.
- Also you can create a daemon for this.
